# MAHJ MAHJ — Marketing Site

Next.js 16 App Router marketing site for [mahjmahj.co](https://mahjmahj.co).

## Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Hosting:** Vercel
- **API:** api.mahjmahj.co (Supabase)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Auto-deploys to Vercel on push to `main`.
